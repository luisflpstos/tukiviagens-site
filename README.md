# Tuki Viagens — Site

Site estático em Astro para hotéis, resorts, destinos, blog e landing pages.

## Stack

- Astro 7 + TypeScript
- Tailwind CSS 4
- Content Collections (Markdown)
- Deploy: Vercel

## Comandos

```sh
pnpm install
pnpm dev
pnpm build
pnpm preview
```

Planejamento completo: [planejamento-mvp-astro-vercel.md](./docs/planejamento-mvp-astro-vercel.md)
